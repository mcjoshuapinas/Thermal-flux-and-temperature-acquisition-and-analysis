Thermal flux and temperature adquisition and analysis

This repository contains a Python-based tool for processing adquistion of thermal flux and temperature using NI modules and NI DADmx drivers.

Features
The script automates data for independant channel at 100 Hz and 100 samples par second.
It provides an automatic real time plot.
It saves data of each time discretisation in a CSV file.

Getting Started
Prerequisites
Install the required libraries:
pip install -r requirements.txt
